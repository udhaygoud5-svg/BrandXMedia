# Supabase Backend Setup Guide

To enable the full-stack functionality (lead collection) for your Unio AI website, follow these steps:

## 1. Create a Supabase Project
1. Go to [supabase.com](https://supabase.com/) and create a new project.
2. Once the project is ready, go to **Project Settings** -> **API**.
3. Copy your `Project URL` and `anon public key`.

## 2. Configure the Website
1. Open `unio-ai-fullstack/app.js`.
2. Replace `YOUR_SUPABASE_URL` and `YOUR_SUPABASE_ANON_KEY` with the values you copied.

## 3. Create the Database Table
Go to the **SQL Editor** in your Supabase dashboard and run the following command to create the `leads` table:

```sql
create table leads (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  name text not null,
  email text not null,
  website text,
  service text
);

-- Enable Row Level Security (RLS)
alter table leads enable row level security;

-- Create a policy that allows anyone to insert leads (for the contact form)
create policy "Enable insert for everyone" on leads for insert with check (true);

-- Optional: Create a policy for you to view leads
-- (In a real app, you'd restrict this to authenticated admins)
create policy "Enable select for authenticated users only" on leads for select using (auth.role() = 'authenticated');
```

## 4. Test the Integration
1. Open `index.html` in your browser.
2. Fill out the "Claim Your AI Audit" form.
3. Check the **Table Editor** in Supabase to see your new lead!
