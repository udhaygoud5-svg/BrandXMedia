# Spline 3D Demo

This folder contains a minimal vanilla‑JS demo that loads a Spline 3D scene.

## How to view
1. **Replace the placeholder** `YOUR_SCENE_ID` in `index.html` with the ID of a scene you exported from Spline.
2. Open the file directly **or** serve it with a local web server so the module import works correctly:
   ```
   # Using Python (already installed on most systems)
   cd c:/Users/satya/Downloads/BrandXMedia/spline-demo
   python -m http.server 8000
   ```
   Then open `http://localhost:8000` in your browser.
3. You should see a loading spinner briefly, then the 3D scene fades in.

## Customisation
- Edit the CSS in the `<style>` block to change colours, layout, or add a background image.
- Use the Spline runtime API (see the skill documentation) to interact with objects, listen for events, or set variables.
